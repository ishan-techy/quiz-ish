# quiz-ish

A fun and interactive command-line quiz package.

## Installation
```sh
$npm install quiz-ish
```

## Usage
```sh
$npx quiz-ish
```

## Dependencies
- [chalk](https://www.npmjs.com/package/chalk)
- [chalk-animation](https://www.npmjs.com/package/chalk-animation)
